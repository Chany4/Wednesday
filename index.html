There are different programming languages that involve web development

HTML

Hypertext Markup Language, or HTML, is a programming language used to describe the structure of web pages.

CSS

CSS is responsible for the text style, size, positioning, color, and more on a website. It's also what controls how a website's style shifts between desktop and mobile versions.

JavaScript

JavaScript is responsible for the interactivity of the web page. It allows the user to interact with the page by by clicking on a button or entering text into a form.
